Introducción a HTML y CSS

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para la creación de páginas web. Define la estructura del contenido mediante etiquetas.

Características principales de HTML:

  • Define la estructura de una página web.
  • Utiliza etiquetas como <h1>, <p>, <a>, etc.
  • Permite incluir imágenes, videos y otros elementos multimedia.
  • Puede integrarse con CSS y JavaScript.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de estilos utilizado para describir la apariencia y el diseño de una página web.

Características principales de CSS:

  • Controla la apariencia visual de los elementos HTML.
  • Permite cambiar colores, fuentes, márgenes, y más.
  • Soporta el diseño responsivo mediante media queries.
  • Facilita la reutilización de estilos en múltiples páginas.
